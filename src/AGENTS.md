# PHOMAX 电商产品展示网站 - 需求拆解文档

## 产品概述

- **产品类型**: 电商产品展示网站（品牌官网/落地页）
- **场景类型**: <scene_type>prototype-app</scene_type>
- **目标用户**: 3C数码电子产品消费者、电池/充电器采购者、电子DIY爱好者
- **核心价值**: 为 PHOMAX 品牌建立专业可靠的线上形象，清晰展示电池与充电器产品线，促进用户了解产品并建立品牌信任
- **界面语言**: 中文
- **主题偏好**: 深色科技风（深蓝色 + 科技蓝渐变，搭配白色和灰色）
- **导航模式**: 锚点导航
- **导航布局**: Topbar（顶部固定，消费者前台品牌官网）

---

## 页面结构总览

> **说明**：品牌官网为单页锚点导航结构，所有内容区块在同一页面内通过锚点跳转

**页面文件**: `HomePage.tsx`

| 区块名称 | 锚点 | 区块说明 |
|---------|------|---------|
| 顶部导航栏 | - | 固定顶部，含 Logo、导航项（首页/产品中心/关于我们/联系我们） |
| Hero 横幅区 | `#hero` | 品牌标语 + 主打产品展示，科技感视觉冲击 |
| 产品分类区 | `#products` | 按充电器系列、电池系列分类展示，含分类筛选 |
| 热门产品展示区 | `#featured` | 6-8 个精选产品卡片，含图片、名称、价格、特点 |
| 品牌优势区 | `#advantages` | 四大核心卖点：品质保证、快速充电、安全可靠、环保节能 |
| 关于我们 | `#about` | 公司简介/品牌故事 |
| 页脚 | `#contact` | 联系方式、版权信息、社交媒体链接 |

## 页面布局建议

- **布局模式**: 单栏全宽滚动（各区块纵向堆叠，Hero 区可做全屏视觉）
- **视觉重心**: Hero 区（品牌第一印象）→ 产品展示（核心转化）→ 品牌优势（信任建立）
- **结果承载区**: 产品卡片网格（热门产品区）和分类筛选结果区，初始态展示全部产品；分类筛选切换时产品卡片区动态更新

## 功能列表

- **区块: 顶部导航栏**
  - **页面目标**: 提供全局导航入口，强化品牌识别
  - **功能点**:
    - **品牌 Logo 展示**: 左侧显示 PHOMAX Logo，点击回到页面顶部
    - **锚点导航**: 首页/产品中心/关于我们/联系我们，点击平滑滚动到对应区块
    - **滚动状态变化**: 页面滚动超过 Hero 区后，导航栏背景由透明变为深色半透明毛玻璃效果
    - **移动端汉堡菜单**: 小屏设备导航项折叠为汉堡图标，点击展开下拉菜单

- **区块: Hero 横幅区**
  - **页面目标**: 建立品牌第一印象，传递核心价值主张
  - **功能点**:
    - **品牌标语展示**: 大字号主标语（如"PHOMAX — 为每一份能量赋能"）+ 副标语，科技感动画入场
    - **主打产品展示**: 中央或右侧展示一款旗舰产品（如 12 槽智能充电器）的 3D 渲染图或高质量产品图，带微动效（缓慢浮动/光效扫过）
    - **CTA 按钮**: "探索产品"按钮，点击平滑滚动至产品中心区块
    - **背景动效**: 深蓝渐变背景 + 粒子/光点流动动画，呼应电池能量主题

- **区块: 产品分类区**
  - **页面目标**: 让用户快速了解产品线结构，按分类浏览
  - **功能点**:
    - **分类筛选标签**: "全部" / "充电器系列" / "电池系列" 三个筛选按钮，点击切换产品展示
    - **分类卡片展示**: 每个分类一张大卡片，含分类名称、代表产品图、产品数量，点击可跳转或联动筛选
    - **筛选动画**: 产品卡片切换时带淡入淡出过渡动画

- **区块: 热门产品展示区**
  - **页面目标**: 展示精选产品，引导用户深入了解
  - **功能点**:
    - **产品卡片网格**: 响应式网格布局（桌面 4 列 → 平板 2 列 → 手机 1 列），展示 6-8 个产品
    - **产品卡片内容**: 产品图片（背景简洁干净）、产品名称、价格、核心特点标签（2-3 个）
    - **卡片悬停动画**: 鼠标悬停时卡片上浮（translateY -8px）+ 阴影加深 + 图片微缩放，过渡流畅
    - **分类联动**: 与上方分类筛选标签联动，筛选后仅展示对应分类产品

- **区块: 品牌优势区**
  - **页面目标**: 建立品牌信任，突出差异化卖点
  - **功能点**:
    - **四列卖点卡片**: 品质保证、快速充电、安全可靠、环保节能，每列含图标 + 标题 + 简短描述
    - **图标动效**: 图标在进入视口时带渐入 + 微弹跳动画（Intersection Observer 触发）
    - **视觉层次**: 深色背景上使用亮色图标和白色文字，形成对比

- **区块: 关于我们**
  - **页面目标**: 讲述品牌故事，拉近与用户的距离
  - **功能点**:
    - **品牌故事文案**: 简短介绍 PHOMAX 品牌理念、技术实力、市场定位
    - **视觉布局**: 左侧文字 + 右侧品牌形象图（或产品应用场景图），左右分栏布局
    - **数据亮点**: 可选展示关键数字（如"服务全球 XX 国家"、"累计出货 XX 万件"），增强可信度

- **区块: 页脚**
  - **页面目标**: 提供联系方式和法律信息
  - **功能点**:
    - **联系方式**: 邮箱、电话、地址（使用占位信息）
    - **快速链接**: 重复导航项（首页/产品中心/关于我们/联系我们）
    - **版权信息**: "© 2026 PHOMAX. All rights reserved."
    - **社交媒体图标**: 微信/微博/抖音等图标链接（占位）

## 数据共享配置

| 存储键名 | 数据说明 | 使用页面 |
|---------|---------|---------|
| `__global_phomax_products` | 产品列表，类型为 `IProduct[]` | 产品分类区、热门产品展示区 |
| `__global_phomax_activeCategory` | 当前选中的产品分类筛选，类型为 `'all' \| 'charger' \| 'battery'` | 产品分类区、热门产品展示区 |

```ts
interface IProduct {
  /** 产品唯一标识 */
  id: string;
  /** 产品名称 */
  name: string;
  /** 产品分类 */
  category: 'charger' | 'battery';
  /** 产品子分类标签 */
  subCategory: string;
  /** 产品图片 URL */
  imageUrl: string;
  /** 产品价格（人民币） */
  price: number;
  /** 产品核心特点标签（2-4个） */
  features: string[];
  /** 产品详细描述 */
  description: string;
  /** 产品规格参数 */
  specs: Record<string, string>;
  /** 是否为热门推荐 */
  isFeatured: boolean;
}
